Installation
  • BookmarksHome will be installed in your personal profile's folder.
  • After installation, close all running browsers and restart one.
  • In the browser's url field, type or paste the url:
    1. chrome://bookmarkshome/content/bookmarksHome.html
  • and press Enter.
  • You can create a button by filing (i.e. bookmarking) the page in the Personal Toolbar Folder (Mozilla) or Bookmarks Toolbar Folder (Firefox).
  • If you want to have BookmarksHome as your home page, set the corresponding preference to:
    1. chrome://bookmarkshome/content/startup.html

Uninstallation
BookmarksHome does no harm, but you can deinstall it by executing the following steps:

Mozilla/Netscape

  1. Go to your personal profile's chrome folder.
  2. Delete bookmarksHome.jar.
  3. Carefully remove every line with a reference to BookmarksHome from chrome.rdf, i.e. the lines similar to: 
          <RDF:Description about="urn:mozilla:package:bookmarkshome"
		   c:baseURL="jar:file:///[...]/chrome/bookmarksHome.jar!/content/bookmarkshome/"
		   c:locType="profile"
		   c:displayName="BookmarksHome 1.0"
		   c:author="Jeroen Groenenboom"
		   c:name="bookmarkshome"
		   c:extension="true"
		   c:description="turns bookmarks into pretty homepage" />
    and the line: 
    	<RDF:li resource="urn:mozilla:package:bookmarksHome"/>
  4. Carefully remove every line with a reference to BookmarksHome (if present) from the overlayinfo folder, i.e.
    in overlayinfo/browser/content/overlays.rdf: 
          <RDF:li>chrome://bookmarkshome/content/bookmarksHomeOverlay.xul</RDF:li>
    and in overlayinfo/navigator/content/overlays.rdf: 
          <RDF:li>chrome://bookmarkshome/content/bookmarksHomeOverlay.xul</RDF:li>
  5. Type about:config in the location bar and reset all preferences (Right click->Reset) that start with bookmarkshome.
  6. Restart Mozilla.

FireFox

  1. Choose Uninstall BookmarksHome in the Tools->Extensions Window.
  2. Type about:config in the location bar and reset all preferences (Right click->Reset) that start with bookmarkshome.
